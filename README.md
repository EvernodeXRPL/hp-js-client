# Hot Pocket javascript client
Hot Pocket javascript client for NodeJs and Browser.

## Prerequisites
```
npm i -g @vercel/ncc
```

## Publish to npm
```
npm install
npm login
npm run publish
```

## Testing example client
```
cd example
npm install
node client.js
```
